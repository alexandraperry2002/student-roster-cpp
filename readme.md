# Student Roster Management System (C++)

This project is a C++ application that manages a roster of students for a university course. It demonstrates object-oriented programming concepts, structured data handling, and validation.

## Features
- Add and remove students from a roster
- Store student data (ID, name, email, age, course days)
- Validate email addresses and detect invalid entries
- Calculate average days to complete courses
- Filter students by degree program
- Display student records in a structured format

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Arrays and pointers
- Enumerations
- Git
